Que es HTML

HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el código estándar utilizado para estructurar y desplegar páginas web y sus contenidos, como párrafos, imágenes y tablas.

Historia de HTML

HTML (HyperText Markup Language) fue creado por Tim Berners-Lee en 1991 en el CERN para facilitar el intercambio de documentos científicos. Evolucionó desde una primera versión simple con 18 etiquetas a HTML5 (2014), el estándar actual que soporta multimedia y aplicaciones web complejas, pasando por hitos como HTML 3.2 y 4.0.

HTML5

HTML5 es la quinta y última versión del estándar de lenguaje de marcado de hipertexto (HTML), fundamental para estructurar y presentar contenido en la web. Introduce mejoras semánticas, multimedia integrada (video/audio sin plugins) y APIs para aplicaciones web avanzadas, ofreciendo un mejor rendimiento y accesibilidad.
